# 💡 WeKnora - 대규모 언어 모델 기반 문서 이해 및 검색 프레임워크

## 📌 프로젝트 소개
[**WeKnora (위노라)**](https://weknora.weixin.qq.com)는 대규모 언어 모델(LLM)을 기반으로 한 문서 이해 및 시맨틱 검색 프레임워크로, 구조가 복잡하고 내용이 이질적인 문서 시나리오를 위해 설계되었습니다.

이 프레임워크는 모듈식 아키텍처를 채택하여 멀티모달 전처리, 시맨틱 벡터 인덱싱, 지능형 리콜 및 LLM 생성 추론을 결합하여 효율적이고 제어 가능한 문서 질의응답 프로세스를 구축합니다. 핵심 검색 프로세스는 **RAG (Retrieval-Augmented Generation)** 메커니즘을 기반으로 하며, 문맥 관련 조각을 언어 모델과 결합하여 더 높은 품질의 시맨틱 답변을 구현합니다.

**공식 웹사이트:** https://weknora.weixin.qq.com

## ✨ 최신 업데이트
**v0.2.0 버전 하이라이트:**
- 🤖 **에이전트(Agent) 모드**: ReACT 에이전트 모드 추가, 내장 도구, MCP 도구 및 웹 검색 호출 지원, 반복 및 반성을 통한 종합 요약 보고서 제공
- 📚 **다중 유형 지식베이스**: FAQ 및 문서 두 가지 유형의 지식베이스 지원, 폴더 가져오기, URL 가져오기, 태그 관리 및 온라인 입력 기능 추가
- ⚙️ **대화 전략**: 에이전트 모델, 일반 모드 모델, 검색 임계값 및 프롬프트 설정 지원, 다중 턴 대화 동작의 정밀 제어
- 🌐 **웹 검색**: 확장 가능한 웹 검색 엔진 지원, 내장 DuckDuckGo 검색 엔진
- 🔌 **MCP 도구 통합**: MCP를 통한 에이전트 능력 확장 지원, 내장 uvx 및 npx 실행 도구, 다양한 전송 방식 지원
- 🎨 **새로운 UI**: 대화 인터페이스 최적화, 에이전트 모드/일반 모드 전환 지원, 도구 호출 과정 전시, 지식베이스 관리 인터페이스 전면 업그레이드
- ⚡ **코어 업그레이드**: MQ 비동기 작업 관리 도입, 데이터베이스 자동 마이그레이션 지원, 빠른 개발 모드 제공

## 🔒 보안 선언
**중요 사항:** v0.1.3 버전부터 WeKnora는 시스템 보안 강화를 위해 로그인 인증 기능을 제공합니다. 운영 환경 배포 시 다음 사항을 강력히 권장합니다:
- WeKnora 서비스를 공용 인터넷이 아닌 내부 망 또는 사설 네트워크 환경에 배포하십시오.
- 중요한 정보 유출 위험을 방지하기 위해 서비스를 공용 인터넷에 직접 노출하지 마십시오.
- 배포 환경에 적절한 방화벽 규칙과 액세스 제어를 설정하십시오.
- 보안 패치와 개선 사항을 적용하기 위해 정기적으로 최신 버전으로 업데이트하십시오.

## 🏗️ 아키텍처 설계
![architecture](./docs/images/architecture.png)
WeKnora는 현대적인 모듈식 설계를 채택하여 완전한 문서 이해 및 검색 파이프라인을 구축했습니다. 시스템은 주로 문서 파싱, 벡터화 처리, 검색 엔진 및 대규모 언어 모델 추론과 같은 핵심 모듈로 구성되며, 각 구성 요소는 유연하게 설정 및 확장할 수 있습니다.

## 🎯 핵심 기능
- **🤖 에이전트 모드**: ReACT 에이전트 모드 지원, 내장 도구 검색, MCP 도구 및 웹 검색 호출 가능, 여러 단계의 반복과 반성을 통해 종합적인 결과 도출
- **🔍 정밀 이해**: PDF, Word, 이미지 등 문서의 구조화된 내용 추출 지원, 통합 시맨틱 뷰 구축
- **🧠 지능형 추론**: LLM을 활용한 문서 문맥 및 사용자 의도 이해, 정밀 질의응답 및 다중 턴 대화 지원
- **📚 다중 유형 지식베이스**: FAQ 및 문서 유형 지원, 폴더/URL 가져오기, 태그 관리 및 온라인 직접 입력 지원
- **🔧 유연한 확장**: 파싱, 임베딩, 리콜에서 생성까지 전 과정이 분리되어 있어 유연한 통합 및 사용자 정의 확장 가능
- **⚡ 효율적 검색**: 키워드, 벡터, 지식 그래프를 결합한 하이브리드 검색 전략, 교차 지식베이스 검색 지원
- **🌐 웹 검색**: 확장 가능한 웹 검색 엔진 지원, 기본 DuckDuckGo 엔진 탑재
- **🔌 MCP 도구 통합**: MCP를 통해 에이전트의 능력을 쉽게 확장
- **⚙️ 대화 전략**: 모드별 모델 설정, 검색 임계값 및 프롬프트의 정밀한 제어
- **🎯 간편한 사용**: 직관적인 웹 인터페이스와 표준화된 API 제공으로 기술적 장벽 최소화
- **🔒 보안 및 제어**: 로컬 및 사설 클라우드 배포 지원, 데이터 주권 완전 보장
