# 03. 지식 관리: 지식베이스 및 문서 업로드

AI가 답변할 기초 자료가 되는 지식베이스를 구축하는 방법을 설명합니다.

## 1. 지식베이스 생성
1. 왼쪽 메뉴의 **"지식베이스" (Knowledge Base)** 탭을 선택합니다.
2. **"지식베이스 생성"** 버튼을 클릭합니다.
3. 지식베이스의 유형을 선택합니다:
   - **문서(Document)**: PDF, Word, TXT, Markdown 등 일반 파일용.
   - **Q&A(FAQ)**: 질문과 답변 쌍으로 구성된 구조화된 데이터용.
4. 이름과 설명을 입력하고 사용할 **임베딩 모델**을 선택한 후 생성을 완료합니다.

## 2. 데이터 업로드
생성된 지식베이스 상세 페이지에서 데이터를 추가할 수 있습니다.

### 문서형 (Document)
- **파일 업로드**: 파일을 드래그 앤 드롭하거나 클릭하여 업로드합니다.
- **웹페이지 가져오기**: 특정 웹사이트의 URL을 입력하여 텍스트를 추출합니다.
- **수동 생성**: Markdown 에디터를 사용하여 직접 내용을 작성합니다.

### Q&A형 (FAQ)
- **개별 추가**: 질문, 유사 질문, 답변을 직접 입력합니다.
- **일괄 가져오기**: Excel이나 CSV 파일을 사용하여 대량의 Q&A 데이터를 업로드합니다.

## 3. 처리 상태 확인
업로드된 파일은 **"파싱 중" -> "완료"** 순으로 상태가 변경됩니다. 상태가 "완료"가 되면 AI 채팅에서 해당 내용을 참조할 수 있습니다.
