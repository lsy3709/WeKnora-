# 03. 지식베이스 구축 가이드

지식베이스는 사용자의 문서(PDF, Word, TXT 등)를 AI가 학습하고 검색할 수 있도록 구조화하여 저장하는 공간입니다.

## 1. 지식베이스 생성
1. 왼쪽 사이드바에서 **지식베이스 (Knowledge Base)** 메뉴를 클릭합니다.
2. 상단의 **지식베이스 생성** 버튼을 클릭합니다.
3. 지식베이스의 **이름**과 **설명**을 입력합니다.
4. 해당 지식베이스에서 사용할 **임베딩 모델**을 선택합니다. (한 번 설정하면 변경이 어려우니 주의하세요)

## 2. 문서 업로드
생성된 지식베이스를 클릭하여 상세 페이지로 이동한 뒤 문서를 추가합니다.

- **파일 업로드**: 문서를 드래그 앤 드롭하거나 클릭하여 선택합니다.
- **웹페이지 가져오기**: URL을 입력하면 시스템이 내용을 자동으로 추출합니다.
- **수동 생성**: 직접 Markdown 형식으로 내용을 작성할 수 있습니다.

## 3. 문서 처리 상태 확인
문서를 업로드하면 시스템이 자동으로 다음 과정을 수행합니다.
1. **파싱 (Parsing)**: 문서에서 텍스트와 이미지 추출
2. **청킹 (Chunking)**: 검색하기 적절한 크기로 텍스트 분할
3. **인덱싱 (Indexing)**: 임베딩 모델을 이용해 벡터 데이터베이스에 저장

상태가 **"완료 (Completed)"**로 변경되면 AI와 대화할 준비가 된 것입니다.
