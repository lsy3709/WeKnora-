# 04. 대화 및 AI 에이전트 활용 가이드

문서 처리가 완료되었다면, 이제 AI와 대화하며 지식을 검색하고 활용할 수 있습니다.

## 1. 대화 시작하기
1. 왼쪽 메뉴에서 **대화 (Chat)**를 클릭합니다.
2. 상단의 **새 대화 (New Chat)** 버튼을 클릭하여 세션을 생성합니다.
3. 질문할 **지식베이스를 선택**합니다. (여러 개를 동시에 선택할 수 있습니다)

## 2. 두 가지 대화 모드

### A. 일반 모드 (Normal Mode)
- **특징**: 선택한 지식베이스에서 관련 내용을 찾아 답변합니다. (전형적인 RAG 방식)
- **용도**: 특정 문서 내의 사실 관계 확인, 요약 등

### B. 에이전트 모드 (Agent Mode)
- **특징**: AI가 스스로 계획을 세우고 여러 도구를 사용합니다. (사고 과정 표시)
- **용도**: 웹 검색 결합, 복잡한 추론, 여러 지식베이스 교차 검색 등
- **설정**: 에이전트 모드를 사용하려면 설정에서 "사고 모델 (Thinking Model)"이 지정되어 있어야 합니다.

## 3. 답변 도구 활용
답변 영역에서는 다음과 같은 편의 기능을 제공합니다.
- **출처 보기**: AI가 어떤 문서를 참조했는지 확인 (클릭 시 해당 문서의 청크 위치로 이동)
- **복사 및 다시 생성**: 답변 내용을 복사하거나 만족스럽지 않을 때 다시 생성 요청
- **웹 검색 토글**: 실시간 인터넷 정보가 필요할 때 켜고 끌 수 있음
