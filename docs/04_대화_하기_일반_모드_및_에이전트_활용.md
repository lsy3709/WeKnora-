# 04. 대화 하기: 일반 모드 및 에이전트 활용

준비된 지식을 바탕으로 AI와 대화하고 고급 기능을 사용하는 방법을 설명합니다.

## 1. 채팅 시작하기
1. 왼쪽 메뉴의 **"대화" (Chat)** 탭을 클릭합니다.
2. 상단의 **"새 대화"**를 클릭하여 세션을 생성합니다.
3. 질문 입력창 상단에서 참조할 **지식베이스**를 하나 이상 선택합니다.

## 2. 일반 모드 (Normal Mode)
- **특징**: 선택한 지식베이스에서 가장 관련 있는 내용을 찾아 답변을 생성합니다 (RAG 방식).
- **활용**: 특정 문서의 내용을 요약하거나 사실 관계를 확인할 때 유리합니다.

## 3. 에이전트 모드 (Agent Mode)
- **특징**: AI가 스스로 문제를 해결하기 위해 계획을 세우고 도구(웹 검색, 지식 검색 등)를 사용합니다.
- **활용**: 복잡한 추론이 필요하거나, 여러 지식베이스를 동시에 탐색하고 웹의 실시간 정보까지 결합해야 할 때 사용합니다.
- **설정**: 에이전트 모드를 사용하려면 "시스템 설정"에서 **사고 모델(Thinking Model)**이 구성되어 있어야 합니다.

## 4. 고급 기능
- **웹 검색 (Web Search)**: 지식베이스에 없는 최신 정보가 필요할 때 채팅창의 웹 검색 스위치를 켭니다.
- **참조 확인**: AI의 답변 하단에 표시된 숫자를 클릭하면, 어떤 문서의 어느 부분을 참고했는지 직접 확인할 수 있습니다.
- **답변 다시 생성**: 답변이 만족스럽지 않다면 "다시 생성" 버튼을 눌러 다른 답변을 유도합니다.
