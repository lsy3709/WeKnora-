# 01. 접속 및 로그인 가이드

이 문서는 WeKnora 웹 브라우저 인터페이스에 처음 접속하고 로그인하는 방법을 안내합니다.

## 1. 서비스 접속
도커 컨테이너가 정상적으로 실행되었다면, 웹 브라우저를 열고 다음 주소로 접속합니다.

- **주소**: `http://localhost` (기본 설정 시)
- 만약 포트를 변경했다면 `.env` 파일의 `FRONTEND_PORT`에 설정한 포트로 접속하세요.

## 2. 회원가입 (처음 사용 시)
WeKnora는 멀티 테넌시를 지원하며, 첫 접속 시 계정을 생성해야 합니다.

1. 로그인 화면 하단의 **"지금 가입하기 (Register Now)"** 링크를 클릭합니다.
2. 사용자 이름, 이메일, 비밀번호를 입력합니다.
3. 가입이 완료되면 시스템이 사용자만의 독립적인 **테넌트 공간**을 자동으로 생성합니다.

## 3. 로그인
1. 가입한 이메일과 비밀번호를 입력하여 로그인합니다.
2. **로그인 상태 유지 (Remember Me)**를 체크하면 다음 접속 시 편리하게 이용할 수 있습니다.
