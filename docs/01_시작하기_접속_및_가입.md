# 01. 시작하기: 접속 및 계정 생성

WeKnora 웹 인터페이스에 접속하고 첫 계정을 생성하는 방법을 설명합니다.

## 1. 웹 브라우저 접속
도커 컨테이너가 정상적으로 실행 중이라면, 선호하는 웹 브라우저를 열고 다음 주소로 접속합니다.
- **주소**: `http://localhost` (또는 설정한 `FRONTEND_PORT`)

## 2. 계정 생성 (회원가입)
첫 접속 시 로그인 화면이 나타납니다. 계정이 없다면 먼저 회원가입을 진행해야 합니다.
1. 화면 하단의 **"지금 가입하기" (Register Now)** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - **사용자명**: 시스템에서 사용할 이름
   - **이메일**: 로그인 아이디로 사용될 이메일 주소
   - **비밀번호**: 8~32자 사이의 문자 및 숫자 조합
3. 가입이 완료되면 시스템이 자동으로 전용 **테넌트(Tenant)** 공간을 생성합니다.

## 3. 로그인
1. 가입한 이메일과 비밀번호를 입력하고 **"로그인"** 버튼을 클릭합니다.
2. 로그인 성공 후 메인 대시보드로 진입합니다.
